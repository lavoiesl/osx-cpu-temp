# OSX CPU Temp

Outputs current CPU temperature in °C for OSX

## Usage 

### Compiling
```bash
make
```

### Running

```bash
./osx-cpu-temp
```

or

```bash
sudo make install # installs to /usr/local/bin
osx-cpu-temp
```

### Output example

```
61.8°C
```
